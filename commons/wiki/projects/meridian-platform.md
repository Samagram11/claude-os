# Meridian Platform
**Status:** Production (v3.2)
**Lead:** [[lin-zhang]]
**Team:** [[james-wright]] + 4 other engineers

## Overview
Meridian's core product — an enterprise analytics platform that provides real-time dashboards over operational data. Used by 50 customers across healthcare, fintech, and logistics verticals.

## Key Features
- Real-time dashboards with sub-second refresh
- Custom dashboard builder
- SSO/SAML integration
- Data connectors for 20+ sources
- API access for programmatic queries

## Frequently Requested Features
1. **Smart alerts / rules-based alerting** — most requested by enterprise customers (see [[smart-alerts]])
2. Scheduled report delivery
3. Dashboard embedding (iframe support)
4. Custom metrics builder

## Recent Development
- Monthly company hackathon produces prototype features
- [[james-wright]] built a working [[smart-alerts]] prototype at the March hackathon
- Hackathon projects live in feature branches and need product approval to ship
